# AI README Generator (FastAPI + Groq)

An intelligent README generator web application built using **FastAPI**, **Groq LLM**, and a **modern HTML/JS frontend**.  
The tool allows users to upload entire project folders, automatically analyzes each file using AI, and generates a polished `README.md` with structured technical documentation.
